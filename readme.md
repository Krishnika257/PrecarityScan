# 🔍 PrecarityScan

> An AI-powered analytics platform for identifying and measuring employment precarity among gig and informal workers through data-driven risk assessment and interactive visualizations.

## Overview

PrecarityScan helps researchers, policymakers, and workers analyze employment insecurity using structured workforce data. The platform computes a **Precarity Score** based on employment conditions, social security benefits, income stability, and work patterns, enabling informed decision-making and policy analysis.

Built with **Python**, **Streamlit**, and **Pandas**, the application provides an intuitive dashboard with role-based access and insightful analytics.

## Features

-  Secure role-based authentication

-  Automatic Precarity Score calculation
-  Interactive dashboards and visualizations

-  CSV-based dataset management

-  Separate views for Researchers, Policymakers, and Workers

## Precarity Score

The platform evaluates employment vulnerability using factors such as:

- Monthly income
- Working hours
- Experience
- Health insurance availability
- Paid leave
- Retirement benefits
- Multiple platform dependency
- Employment loans
- Weekly working days

These indicators are combined into a comprehensive **Precarity Score** to identify levels of employment insecurity.

##  Motivation

Millions of gig workers face unstable employment, uncertain income, and limited social protection. **PrecarityScan** aims to make employment precarity measurable through data analytics, empowering researchers, policymakers, and workers with actionable insights for better decision-making.
